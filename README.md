Exploración de datos de SINADEF relacionados a muertes violentas por posible
arma de fuego, homicidio, etc.

Datos de: https://files.minsa.gob.pe/s/Ae52gBAMf9aKEzK (descargados el 2025/12/15)

Algunos de los datos en formato CSV están comprimidos usando XZ que R puede leer y escribir nativamente, para que los archivos puedan ser puestos sin problema en github.
